# DB Lab 1 – SQL (DDL & DML)

## Overview
This repository contains solutions for DB Lab 1 covering SQL Data Definition Language (DDL) and Data Manipulation Language (DML) operations.

## File Structure
- `lab1_solution.sql` – All SQL queries for Lab 1
- `questions.txt` – Lab questions
- `README.md` – Instructions and overview

## How to Execute
1. Open Oracle SQL Developer / MySQL Workbench.
2. Open `lab1_solution.sql`.
3. Execute statements step-by-step or run the script.
